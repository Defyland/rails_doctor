# Roadmap

1. More validated policy-file examples for additional deploy topologies and managed platforms.
2. Chat, webhook, and non-GitHub downstream integration workflows on top of suppression reports and upcoming expiry.
3. Broader automatic hook coverage beyond `server`, `db:migrate`, `db:prepare`, `db:schema:load`, `db:structure:load`, and `assets:precompile`, but only where the signal stays actionable.
4. Deeper dependency readiness coverage for more third-party adapters and real client/config shapes beyond the current Sidekiq and GoodJob helpers.
5. More real-world validation against representative production Rails apps.
